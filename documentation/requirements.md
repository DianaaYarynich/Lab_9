# Вимоги до проєкту

## Функціональні вимоги
1. Система повинна підтримувати реєстрацію користувачів
2. Користувачі можуть створювати та редагувати документи
3. Підтримка спільної роботи в реальному часі

## Нефункціональні вимоги
- Продуктивність: час відгуку < 2 секунди
- Безпека: шифрування даних
- Сумісність: підтримка Chrome, Safari
